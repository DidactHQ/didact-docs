# Migrate

Runs migrations against Didact's metadata database.

```bash-vue
didact migrate
```

::: danger
This command requires that you set the [database provider key](/core-concepts/architecture/metadata-database#database-providers) and database connection string in the config. See [config](/core-concepts/setup/config) for more details.
:::

## Examples

Run migrations.

```bash
didact migrate
```
