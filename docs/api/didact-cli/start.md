# Start

Launches the specified Didact application as a new process.

```bash-vue
didact start <app>
```

where `<app>` is the Didact application to start, either `engine` or `ui`.

## Examples

Launches Didact Engine as a new process.

```bash
didact start engine
```

Launches Didact UI as a new process.

```bash
didact start ui
```
