# UI Installation

Here we review the various ways that you can install Didact UI.

## Recommended Installation

The recommended way to install Didact UI is to use the Didact CLI. See the [ui install command](/api/didact-cli/ui-install) for details.

## Manual Installation

If necessary, you can manually install Didact UI - though it is not the recommended path.

To manually install Didact UI, you can download the binary from the [Releases page](https://github.com/DidactHQ/didact/releases) of the main [Didact repository](https://github.com/DidactHQ/didact) on GitHub.

::: danger
However, you must be sure to match up the version of Didact UI to the corresponding version of Didact CLI that you are using! Unsanctioned version mismatches can cause unexpected behavior in the platform, so proceed at your own risk.
:::
