# Engine Installation

Here we review the various ways that you can install Didact Engine.

## Recommended Installation

The recommended way to install Didact Engine is to use the Didact CLI. See the [engine install command](/api/didact-cli/engine-install) for details.

## Manual Installation

If necessary, you can manually install Didact Engine - though it is not the recommended path.

To manually install Didact Engine, you can download the binary from the [Releases page](https://github.com/DidactHQ/didact/releases) of the main [Didact repository](https://github.com/DidactHQ/didact) on GitHub.

::: danger
However, you must be sure to match up the version of Didact Engine to the corresponding version of Didact CLI that you are using! Unsanctioned version mismatches can cause unexpected behavior in the platform, so proceed at your own risk.
:::
