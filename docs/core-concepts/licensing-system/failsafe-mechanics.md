# Failsafe Mechanics

::: warning
The failsafe mechanics only pertain to Didact customers with license keys. If you are not a Didact customer and do not have a license key, then you can ignore this page.
:::

In case of a licensing system outage, Didact's platform can revert to a failsafe mode. The failsafe mode operates in such a way so that you do not lose access to enhanced features during an outage; instead, you can enjoy a grace period of uninterrupted use while the licensing system is restored.

This is possible thanks to the [smart key system](/core-concepts/licensing-system/smart-keys).
